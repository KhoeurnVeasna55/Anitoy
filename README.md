## 🚀 Features

- User login & registration  
- Browse products by category  
- Product detail pages  
- Add to cart / update quantity  
- Select items to checkout  
- Checkout form (name, email, address, etc.)  
- Upload payment slip  
- Search bar  
- Admin panel for managing products & orders
